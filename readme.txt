Music Information Retrieval library for Processing.

INSTALL

This repository is organized so that you can place it in a folder named Audio-Descriptors inside of your Processing sketchbook's libraries folder.

* Download the zip file at https://github.com/d-t/Audio-Descriptors/blob/master/distribution/AudioDescriptors/download/AudioDescriptors.zip
* Extract the contents of that package into a directory named Audio-Descriptors inside of the libraries folder in your sketchbook.
